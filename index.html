To use my sudoku solver follow the instructions

Instructions:

1- Open the console with f12 or left click and then 'developer tools'.

2- in the console write a new variable: let newSudoku = [[Here you insert your Sudoku]].

Example: let newSudoku1 = [[1, 5, 0, 0, 4, 2, 0, 0, 6], [2, 7, 4, 5, 6, 0, 0, 1, 0], [0, 0, 6, 0, 0, 7, 4, 0, 2], [0, 1, 0, 0, 0, 0, 0, 4, 0], [0, 0, 0, 0, 5, 0, 0, 0, 0],
[0, 6, 0, 4, 0, 3, 1, 9, 0], [0, 2, 0, 6, 0, 5, 9, 0, 0], [9, 8, 5, 0, 3, 0, 0, 6, 0], [0, 4, 0, 2, 1, 9, 8, 3, 0]];

Every opening of square brackets is a new Sudoku row.

3- Then click enter, your sudoku variable is created, you should get undefined

4- To solve the sudoku write this in the console: solveIt("your sudoku name");. You should get "true".

5- To see the completed Sudoku write: "console.log(sudoku)";. You should get an array, click on it ... AND THEN...

6- Magic!! Your sudoku is solved! Hope you enjoyed it and don't abuse this! Because sudokus are also fun to do without algorithms!
